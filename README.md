# apidoc
API Rest Documentation Generator

## Usage

**Installation**
```bash
npm install
```
```bash
mongoimport --db users --collection settings --file settings.json
```

**Runing**
```bash
npm start
```
Runing with node
```bash
node index.js
```

Runing with nodemon
```bash
nodemon index.js
```
Open in your browser

```bash
http://localhost:8080
```
